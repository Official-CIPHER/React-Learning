02 Import and Export


2 types of export


1st - Default Export

    -> Only one Default export per file.
    -> Import name can be anything.
    -> No {}
  
    Code:- app.js
    const a = 10;
    export default a;
    
    /script.js
    import importfromapp from "./app.js"
    console.log(importfromapp);

    Output - 10

2nd - Named Export

    -> You can multiple export per file.
    -> Must be used same name
    -> Use {}
    -> Safer for large project.
  
    Code:- app.js
    export const b = 120;
    
    /script.js
    import {b} from "./app.js"
    console.log(b);

    Output - 120
